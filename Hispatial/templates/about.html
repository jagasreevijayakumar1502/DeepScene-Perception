About HiSpatial

The HiSpatial Environmental Perception Dashboard is a Flask-powered research demo for hierarchical spatial reasoning.

It combines segmentation, classification, depth estimation, scene graph construction, and natural language generation into a modern visual analytics interface.

  • Object segmentation via SAM-style processing
  • Object classification using CLIP-style vision-language matching
  • Monocular depth estimation and 3D coordinate generation
  • Scene graph generation with spatial relations
  • HiSpatial hierarchical reasoning layers
  • Natural language relationship generation
